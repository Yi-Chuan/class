面向对象第一天
一,学习目标
	1,语法的学习
	2,面向对象的编程思想学习(不是一下子就能转变过来的,慢慢来,例如学数学,一开始觉得很难,后面学会了觉得很简单)
二,面向过程和面向对象 对比
	1,面向过程 (单身屌丝)
		饿了,	自己做饭吃
		渴了,	自己去烧水
		衣服脏了,	自己去洗衣服
		空虚寂寞了,	自己玩LOL
	2,面向对象 (有对象了,长得像朱茵)(解决需求时完全不一样了,不需要什么事情都自己去做了)
		饿了,	朱茵去做饭
		渴了,	朱茵去烧水
		衣服脏了,	朱茵去洗衣服
		空虚寂寞了,	朱茵和我一起玩LOL
		举例:	开店的小故事
三,对象
		具有一定功能和特征的事物 就是 对象
		能满足某种需求的事物 都是 对象
		
四,类
		具有相同功能和特征的对象 的抽象 就是类
		类这个东西是思想上的
		
		*类的实例化(实体化)称之为对象
		*对象的抽象就是类
五,类比
	马良     画的画   从画里走出来的事物
	程序员     类 		对象
	设计师 	  图纸    按图纸造出来的实物
	
六,类的声明(先有类,才有对象)
	a)格式  class 类名 {}   
	b)成员属性
		i.成员属性变量 必须要加修饰符(public protected private)
		ii.成员属性的初始值可有可无
		iii.成员属性在赋值的时候不能使用变量计算以及函数调用
		iv.可以使用常量来进行赋值
	c)常量的值 只有可能是五种  
		Boolean	int	float	string	null
	d)成员方法
		i.成员方法修饰符可有可无
		ii.参数也是可有可无
		iii.函数体和返回值也是可有可无
	e)如果在函数的内部没有使用return手动返回一个值的话 默认返回null
	f)在php中将值变成false的情况 
		false 0  0.0 ‘’’0’ array  null
	g)页面乱码的检测位置
		i.数据库
		ii.页面输出 (header  meta) header(‘Content-type:text/html;charset=utf-8’);
		iii.文件编码
	h)对象的实例化
		i.格式  $var = new 类名();
		ii.对象的实例化 可以在类的声明之后 也可以在声明之前

			object  =>  对象(数据类型)
			MeiNv  => 类名
			[1]   =>  当前对象在脚本中的所有对象的一个数字索引
		iii.成员属性的操作
			$obj->name;
		iv.unset函数没有返回值
			v.云笔记   有道  麦库  印象

七.$this关键字
		this单词本身就是 这个 的意思
		$this在成员方法中代表的是当前的这个对象
		那个对象来调用当前这个方法 $this就代表哪个对象
		当前方法成员访问符前面的对象是谁 $this就代表谁
		
		
八.构造方法
	a)构造方法是一个特殊的成员方法
	b)构造方法 不需要成员访问符去调用
	c)有自己自动的执行场景
	d)当使用new关键字实例化对象的时候会自动触发构造方法
	e)方法名 : __construct  与类名同名的成员方法也是构造方法
	f)作用: 初始化当前这个对象
	
九.析构方法
	a)当对象在销毁的时候  自动执行
	b)方法名:  __destruct
	c)销毁的几种情况
	i.   当脚本执行完毕之后
	ii.   Unset
	iii.重新赋值,覆盖性的
