面向对象第二天

1.封装
	解释: 就是使用修饰符来修饰成员属性和成员方法,
		最大限度的隐藏对象的内部细节,
		保证对象的完整性和安全性.

修饰符:protected  受保护的   
	private   私有的

	i.在类的外部不能直接通过成员访问符来操作对象的受保护和私有的成员属性

	ii.在类的外部不能直接调用当前对象的受保护和私有的方法

	iii.在类的内部是可以直接操作任意修饰符修饰的成员属性和成员方法

b)魔术方法
	i.自动执行的成员方法就是魔术方法
	ii.music =>  gusic   => 
1.g => __get
	触发场景: 当对对象中的非公有的属性或者是不存在的属性进行的获取的时候 会自动触发
2.u=> __unset
	触发场景: 当使用unset函数去销毁对象中的非公有属性或者是不存在的书  会自动触发
3.s => __set
	触发场景: 当在类的外部对对象的非公有属性或者不存在的属性进行赋值的时候 会触发
4.i=>__isset
	触发场景: 当使用isset函数去检测对象中非公有属性或者不存在的属性的时候  自动触发
5.c=> __call
	call 调用
	触发场景:当调用对象中非公有方法的时候 会自动触发该魔术方法
2.继承
	继承的使用场景:
	当别人有  自己没有 继承那个人  叫他一声爹
	继承的格式
	class A extends B{}
	当A继承B类之后  A类中将拥有B类中所有的成员属性和成员方法(除了私有成员之外)

	访问权限
 	              public    protected    private
	在类的外部       √          X           X
	在子类中         √          √           X
	在本类中         √          √           √

	子类方法对父类方法的重写

	如果子类中拥有一个跟父类同名的方法的话,实例化子类对象调用该方法, 调用的是子类的方法 而不是父类的

	php继承只支持单继承
