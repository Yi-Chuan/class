1.  FSCICATS

2.i  =>  instanceof
a)是一个运算符
b)用来检测当前这个对象是否为该类或者该类子类(后辈的类)的实例化
3.c  =>  clone
a)是一个关键字
b)用来复制克隆对象
c)格式  $a = clone  $b;
d)魔术方法  __clone 
i.自动触发场景: 当使用clone关键字复制一个对象的时候 会自动触发该对象的__clone魔术方法
4.a  =>  __autoload  auto自动的意思   load加载的意思
a)这是一个函数
b)自动触发场景:  当使用new关键字实例化一个不存在的类 或者是继承一个不存在的类的时候 自动触发
c)  参数: 当前不存在的类名(字符串)
d)  作用: 为了实现类的自动加载
5.t  => __toString   to转变成  string字符串
a)是一个魔术方法
b)执行场景:  当在脚本中直接对对象进行字符串操作的时候 自动触发
c)   这个方法必须要返回一个字符串.
d)  作用:  是代替对象进行字符串的操作.
6.s  =>  serialize  和unserialize
serialize (串行化  序列化)
a)是一个函数
"a:5:{i:0;s:9:"玩游戏";i:1;s:9:"打篮球";i:2;s:9:"踢足球";i:3;s:6:"台球";i:4;s:9:"羽毛球";}"

a =>  array数组
5 =>  元素的个数
i  => 键名类型
0  => 键名下标
s  => 键值的类型
9  => 键值的长度
玩游戏  =>  键值

"O:8:"SuperMan":3:{s:4:"name";s:6:"肯特";s:6:"height";s:5:"185cm";s:6:"weight";s:4:"80kg";}"

o  =>  object对象
8  =>  类名的长度
SuperMan  => 类名
3 =>   对象属性的个数
s  =>  属性名的类型
4  => s属性名的长度
name =>属性名
s =>  属性值的类型
6  => 属性值的长度
肯特 => 属性值

b) __sleep
是一个魔术方法
当使用serialize进行串行化对象的时候 自动执行
作用 是要筛选要保留的属性的名称.
注意点:  必须要返回一个数组.
c)  __wakeup
是一个魔术方法
当使用unserialize反串行化一个对象的时候 自动执行
作用: 初始化反出行花出来的对象

应用场景: 缓存
